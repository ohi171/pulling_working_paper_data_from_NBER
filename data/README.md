# Sample Data

`sample_papers.csv` contains 40 records selected at evenly spaced positions from the structured NBER Environment and Energy Economics dataset. It is included so the knowledge-graph tutorial and offline smoke test can run without fetching the full corpus.

The file contains bibliographic metadata, NBER source links, and short structured notes. It does not contain full paper abstracts or PDFs. NBER and paper content remain subject to their respective owners' terms and are not covered by this repository's MIT License.
